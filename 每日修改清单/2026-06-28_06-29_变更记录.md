## 🧠 算法核心变更总结

### 旧版（四维度）
```
综合分 = 技能匹配×50% + 测评适配×25% + 学习进度×15% + 基础匹配×10%
```

### 新版（五维度）
```
综合分 = 技能匹配×30% + 测评适配×25% + 学习进度×15% + 成果测评×20% + 基础匹配×10%
```

### 评分引擎升级详情

| 维度 | 旧逻辑 | 新逻辑 |
|------|--------|--------|
| **技能匹配** | 权重50%（自报技能为主） | 权重降至30%（降低自报偏差） |
| **测评适配** | 最近1次测评的5维分 | 最新成绩(60%) + 趋势分析(20%) + 等级分(20%) |
| **学习进度** | 简单完成数占比 | 阶段加权(BASIC:0.7→INTERVIEW:1.5) + 时效衰减 |
| **成果测评** | ❌ 不存在 | 🆕 阶段加权平均(50%) + 最新成绩(30%) + 通过率(20%) |
| **基础匹配** | 学历+城市+专业 | 无变化 |
