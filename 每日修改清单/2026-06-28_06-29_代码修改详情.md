    //   1. 按阶段从题库查所有题目
    //   2. Collections.shuffle 随机打乱
    //   3. 取前10题
    //   4. 返回给前端**不含答案**（只返 content + options）
    //   5. 答案存入 Redis (TTL 30分钟)，Key=lr:session:{userId}:{stage}

    // 提交测评：
    //   1. 从 Redis 取答案（兜底→内存 ConcurrentHashMap）
    //   2. 逐题比对：答案为字母，忽略大小写
    //   3. 计算 correctCount, totalScore, 正确率
    //   4. 正确率≥60% passed=1
    //   5. 保存 learning_result 表
    //   6. 清理 Redis session
    //   7. 返回每题详情（含你的答案+正确答案+解析）
}
